# Other commands:

- `:list`: view queue
- `:clear`: clear queue
- `:mov <i> <j>`: move item at index i to index j
- `:del <i>`: remove item at index i from the queue
- `:<i>`: show info about item at index i
- `C-d`: scroll down
- `C-u`: scroll up
- `.` or space: pause
- `<` or left arrow key: previous song
- `>` or right arrow key: next song

- `C-q`, `C-c`, `:q`, `:exit`: quit
